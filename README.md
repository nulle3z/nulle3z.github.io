# README

## 简介

[nulle3z.github.io](nulle3z.github.io) 将采用自主设计的静态网页框架，使用 **HTML5 + CSS + JavaScript** 构建前端功能，使用 **`Python`** 弥补 `Github Pages` 无法提供的服务端功能，在任何设备上只需要该计算机拥有 **Git 命令行工具** 以及 较新的 **Python** 版本即可。

---

## 各页面功能简介

### index.html

部署在 **Github Pages** 上的默认主页。

### album.html

相册（包含：movie、image）。

### blog.html

个人博客（通过 marked.js、github-markdown-css、blog.js 渲染）。
